## 关于程序  
  ### 建议
  建议分辨率：2520x1680  
  建议缩放：150%  
  建议操作系统：Windows 10 21H2（最稳定），Windows 11 22H2（最具反馈价值）  
  注意：在测试的Windows 7 SP1系统中，普通版本的程序无法正常运行
